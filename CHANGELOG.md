# Changelog

## 1.0.0 — 2026-07-14

- เปิดตัว Nan Living OS บนแนวคิด Knowledge-first และ Community-first
- เพิ่ม Landing, Explorer, Knowledge Graph, Knowledge Assistant, Community, Dashboard และ Map
- เชื่อม Supabase PostgreSQL, Auth client, RLS และ provenance citations
- เพิ่มฐานความรู้ภาษาไทย 38 รายการจากแหล่งทางการและแหล่งชุมชน
- เพิ่มพิกัด WGS84 ที่ตรวจสอบได้สำหรับสถานที่ 13 รายการ
- เพิ่ม Starter Experiences 3 รูปแบบที่เริ่มจากเรื่องราว ชุมชน และผลกระทบ
- เพิ่ม responsive design, dark/light themes และ accessibility foundations
- เพิ่ม Backend/Frontend tests, lint checks และ production build validation
